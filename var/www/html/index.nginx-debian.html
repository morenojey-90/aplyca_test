Bienvenidos Aplyca!

Para llevar a cabo la instalación de el servidor Web realicé los siguientes pasos:
1. sudo apt-get update para actualizar los repositorios.
2. sudo apt-get install nginx para instalar el servidor web.
3. nginx -v para comprobar que efectivamente se instaló Nginx y validar la versión.
